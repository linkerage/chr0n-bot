Notes:
------
- Each user has their own independent composition
- Compositions persist across bot restarts
- Notes are logged when played (check ircbot.log)
- Use MIDI note numbers (60 = middle C)
- Beats are relative to the tempo (140 BPM = 140 beats per minute)
- Multiple notes at the same start time create chords
- Track 0 is created by default, add more tracks as needed
